# Workflow Matrix

## The Lifecycle of an Order

1. **Quote**: Created natively without deducting materials. Exists purely for customer approval.
2. **Pending Approval**: Customer notified explicitly reviewing pricing.
3. **Approved**: Officially an Order. Inventory layers mapped cleanly natively triggering standard material reservations seamlessly.
4. **Materials Ordered**: Required materials sent into Purchase Orders via connected Vendor boundaries.
5. **Materials Received**: Vendor confirms supply deliveries mapping securely internally.
6. **In Production**: Explicit physical work is executed on the materials.
7. **Quality Check**: Manual visual inspection bounds.
8. **Ready to Ship / Shipped**: Label generated natively mapping effectively.
9. **Delivered / Completed**: The final resolved terminal status removing reserved hooks gracefully and cleanly finishing the order flow.
