# Deployment Roadmap

## Phase 1 — MVP
- Frontend/Backend: Vercel
- Database: Supabase (PostgreSQL)
- Auth: Clerk
- Billing: Stripe (Test mode)
- Storage: Supabase Storage

## Phase 2 — Growth
- Supabase Pro
- Vercel Pro
- Clerk Pro
- Stripe Live Mode
- Resend (Email), PostHog (Analytics), Sentry (Errors)

## Phase 3 — Scale
- Dedicated PostgreSQL (AWS RDS/Railway)
- Cloudflare Pro
- Vercel Enterprise
